# About this project
This is a full-stack app that allows users to select a focus and receive reminders to stretch at a schedule that they choose.


### View my project here: 

##### Installation

1. Clone repo
2. run `npm install`

##### Usage

1. run `node server.js`
2. Navigate to `localhost:8000`
